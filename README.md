# SimbirSoft Planner — Тестовое задание

Мобильное приложение **ежедневник** для Android, выполненное в рамках тестового задания от компании **SimbirSoft**.

### Что реализовано

- **Главный экран**: календарь + таблица на 24 часа (14:00–15:00)
- **Подробное описание задачи** (второй экран по ТЗ)
- Добавление, **редактирование** и **удаление** задач
- Задачи хранятся в Room + первичные данные загружаются из `tasks.json`
- Красивый Material Design (цветные карточки, иконки, современные кнопки)
- Автоматический выбор сегодняшней даты при запуске

### Технологии

- Kotlin
- Room, ViewModel, LiveData, RecyclerView
- Material Components
- Gson (для парсинга JSON)
- MVVM + Clean Architecture элементы

### Как запустить

1. Клонировать репозиторий
2. Открыть в Android Studio
3. Запустить на эмуляторе или устройстве (min SDK 24+)
